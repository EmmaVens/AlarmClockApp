# Smart Alarm

Android-приложение умного будильника на Java.

## Функции

- Добавление будильников
- Удаление будильников
- Редактирование будильников
- Повтор по дням недели
- Выбор мелодии
- Отключение будильника встряхиванием телефона

## Технологии

- Java
- Android Studio
- XML
- Accelerometer Sensor
- AlarmManager
- SharedPreferences

## Особенность

Будильник отключается только после 5 движений телефоном, похожих на шаги.

## Запуск

Открыть проект в Android Studio и нажать **Run**.
